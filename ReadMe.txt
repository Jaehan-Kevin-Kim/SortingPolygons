Program Description: Sorting Out Sorting is a program that implements six 
sorting Algorithms: Bubble, Heap(our sorting algorithm of choice), 
Insertion, Merge, Quick and Selection. These sorting algoritms take Polygons
from three txt files (polyfor1.txt, polyfor3.txt and polyfor5.txt), compare
each polygon and sort them in descending order. These polygons can be 
compared by height, volume or base area.  

Program Completeness: 100%

HOW TO RUN THE PROGRAM: 
1- Run Command Prompt
Ex.: Windows key --> CMD
2- Change the current directory to the directory which includes Sort.jar and txt files
Ex.: cd  C:\Users\user\CPRG_311_workspace\Assignment_2\
3- Enter commands via Command Line
Ex.: java -jar Sort.jar -fpolyfor1.txt -Ta -Sb

VALID INPUT Examples:
java -jar Sort.jar -fpolyfor1.txt -Ta -Sb
java -jar Sort.jar -fpolyfor3.txt -Tv -Sm
java -jar Sort.jar -fpolyfor5.txt -Ta -Si
java -jar Sort.jar -ta -Si -fpolyfor1.txt
java -jar Sort.jar -Th -Sq -fpolyfor3.txt
java -jar Sort.jar -tv -Ss -fpolyfor5.txt
java -jar Sort.jar -Sb -fpolyfor5.txt -Th
java -jar Sort.jar -Sq -ta -fpolyfor1.txt
java -jar Sort.jar -Sz -tv -fpolyfor3.txt
